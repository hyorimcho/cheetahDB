# 🐆cheetahDB

[tmdb](https://www.themoviedb.org/?language=ko) api 를 사용한 영화 검색 사이트

## 결과 링크

[결과 링크](https://cheetahdb.vercel.app/)는 여기서 확인하실 수 있습니다.

## 기술 스택

- React
- Styled-components
- Vite

## 개발 기간

- 2023.01.08 - 2023.01.13

## 구현 기능

- 메인 페이지
- 영화 상세 페이지

## 팀원
- 유지석
- 조효림

  - 배열과 객체 처리

- 컴포넌트 재사용
  - 재사용 가능한 컴포넌트를 잘 분리하지 못한 점
